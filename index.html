Detect modulation and filter

Detect modulation and filter based on constellation image
This project used deep learning to classify received signal modulation and filter parameters. Signal data and labels are simulated in Mathlab and are collected into training set and testing set. The detection system contains 2 parts. First part is preprocess block that transform signal data into constellation images. Those images and labels are sent to second part (trained Alexnet model). Output of second block is modulation and filter parameter.
I coded project in Python with Tensorflow framework.

Classify apple quality

Classify apple quality by using Kmeans and SVM
The application identifies fruit diseases by fruit picture. The system includes 3 parts: data preprocess, feature extraction, training and classification. Data preprocess detects the infected region by using Kmeans. In next step, Local Binary Patterns is used to extract feature from infected region only. In last step, we use a one vs rest SVM model to classify. This is a project of team 3 people. I'm in charge of data preprocess part. I coded Kmeans algorithm in Python then use it to detect infected region.
